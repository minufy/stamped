# 개요
이 프로젝트는 테트리스 게임을 기본적인 싱글플레이어만 구현해놓은 프로젝트야.

너가 해야할 일은 온라인 멀티플레이어를 구현하는거야.
기존에 존재하는 scenes/menu.lua를 수정하고, 필요시 scene을 추가로 더 만들어도 돼.

# 프로젝트 구조
- assets/ - 에셋
- modules/ - 외부 모듈
- objects/ - 게임오브젝트
- scenes/ - 씬
- stuff/ - 게임 independent한 범용 코드 

# universal rule
- 코드 최대한 분리
- 하드코딩 최소화
- 인간이 나중에 편집하기 좋게 구현
- 작업 전에 너가 할 일을 계획한 체크리스트인 마크다운 파일을 만들어서 나한테 컨펌을 받고 진행
- 마크다운 파일의 완료된 작업은 알아서 체크
- 큰 작업은 한번에 하지 않고, 쪼개서 진행

# 노트
